# SV-TrustEval-C: Evaluating Structure and Semantic Reasoning in Large Language Models for Source Code Vulnerability Analysis

## PaperSearch Summary

本文提出SV-TrustEval-C基准，用于评估大语言模型在C语言源代码漏洞分析中的结构推理和语义推理能力。结构推理考察模型识别代码元素关系的能力，语义推理检验逻辑一致性。实验表明，当前LLM在理解复杂代码关系方面表现不佳，其漏洞分析更多依赖模式匹配而非稳健的逻辑推理。该基准揭示了提升LLM在真实漏洞分析任务中推理能力和可信度的关键方向。初始基准数据集已公开。
