# Assessing the Latent Automated Program Repair Capabilities of Large Language Models using Round-Trip Translation

## PaperSearch Summary

该论文提出利用往返翻译（Round-Trip Translation）方法评估大型语言模型（LLM）在自动程序修复（APR）中的潜在能力。通过将代码翻译为另一种语言再翻译回原语言，诱导LLM生成修复补丁，无需显式训练。实验在多个基准数据集上验证了该方法能有效暴露LLM的修复能力，并分析了不同翻译路径对修复效果的影响。研究揭示了LLM在代码转换过程中隐含的修复知识，为无需微调的APR提供了新思路。
